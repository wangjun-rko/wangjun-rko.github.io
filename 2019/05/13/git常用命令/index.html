git常用命令

May 13, 2019 | 高珺如其他工具

git常用命令

  1. 设置提交代码时的用户信息

    $ git config user.name “name“ $ git config user.email “email“

  2. 在当前目录新建一个仓库

    $ git init

  3. 下载一个仓库

    $ git clone url

  4. 添加当前目录的所有变化文件到暂存区

    $ git add .

  5. 取消添加

    $ git reset HEAD <filename>

  6. 提交到仓库区

    $ git commit

  7. 提交暂存区到仓库区

    $ git commit -am [message]

  8. 修改最后一次提交的信息

    $ git commit –amend -m 替换的message

  9. 列出本地分支

    $ git branch

  10. 列出远端分支

    $ git branch -r

  11. 列出所有分支

    $ git branch -a

  12. 查看当前详细分支信息（可看到当前分支与对应的远程追踪分支）

    $ git branch -vv

  13. 新建一个名为 branch-name 的分支，但依然停留在当前分支

    $ git branch branch-name

  14. 重命名当前[old-branch-name]分支

    $ git branch -m [old-branch-name] new-branch-name

  15. 重命名远程[origin/old-branch-name]分支

    $ git push origin –d old-branch-name

  16. 切换到 branch-name 分支，并更新工作区

    $ git checkout branch-name

  17. 新建并切换到 branch-name 分支

    $ git checkout -b branch-name

  18. 切换到上一个分支

    $ git checkout -

  19. 新建并切换到连接到指定远程分支的 branch 分支

    $ git checkout origin/branch -b branch

  20. 关联指定本地分支与指定远程分支

    $ git branch –set-upstream-to=origin/remote_branch local_branch

  21. 新建当前分支的远程分支

    git push origin new_branch

  22. 删除分支
    1. 只能删除已经被合并的分支

      $ git branch -d branch-name

    2. 强制删除

      $ git branch -D branch-name

    3. 删除远程分支

      $ git push origin –delete branch-name

    4. 删除本地的远程分支

      $ git branch -r -D origin/branch-name

  23. 变基

    $ git rebase [branch]

  24. 合并 branch 分支到当前分支

    $ git merge [branch]

  25. 显示有变更的文件

    $ git status

  26. 显示当前分支的版本历史

    $ git log

  27. 显示 commit 历史，以及每次 commit 发生变更的文件

    $ git log –stat

  28. 搜索提交历史，根据关键词

    $ git log -S keyword

  29. 显示当前分支的最近几次操作

    $ git reflog

  30. 显示指定文件是谁、何时修改过

    $ git blame [file]

  31. 显示暂存区和工作区的差异

    $ git diff

  32. 显示工作区与当前分支最新 commit 之间的差异

    $ git diff HEAD

  33. 显示两次个分支之间的差异（显示文件内容）

    $ git diff first-branch second-branch

  34. 显示所有远程仓库

    $ git remote -v

  35. 查看当前远程仓库信息

    $ git remote -vv

  36. 下载远程仓库的所有变动

    $ git fetch [remote]

  37. 移除所有在远端已经被删除的远程分支

    $ git fetch -p

  38. 上传本地 branch 分支到远程 remote 仓库

    $ git push remote branch

  39. 重置 reset

    $ git reset [–mode] [commit] [file]

    • mode
      • soft：重置暂存区与工作区[的 file]，[与指定 commit 保持一致]，回退点之前的所有信息都在，只是没显示。
      • mixed：默认，重置暂存区[的 file]，[与指定 commit 保持一致]，工作区不变。
      • hard：重置暂存区与工作区[的 file]，[与指定 commit 保持一致]，回退点之前的所有信息都删掉。
  40. 储藏所有变更

    $ git stash -a

  41. 储藏所有变更（带备注）

    $ git stash -a save ‘message‘

  42. 查看现有的储藏

    $ git stash list

  43. 恢复最近一次储藏，存储仍在栈上

    $ git stash apply

  44. 恢复指定储藏，存储仍在栈上

    $ git stash apply stash@{num}

  45. 删除指定储藏

    $ git stash drop stash@{num}

  46. 删除所有储藏

    $ git stash clear

  47. 恢复并删除栈上指定储藏

    $ git stash pop stash@{num}

  48. 查看当前仓库中的所有未打包的objects和磁盘占用

    $ git count-objects –human-readable

  49. 从object数据库中删除所有不可达的object

    $ git gc –prune=now –aggressive
